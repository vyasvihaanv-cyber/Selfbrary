# Selfbrary

## Project Documentation

Selfbrary is a personal library management website that helps users manage their personal book collections efficiently. It provides features such as adding, removing, and searching for books, as well as tracking loaned books.

### Features
- **User Authentication**: Secure login for users to manage their libraries.
- **Book Management**: Add, edit, and delete book entries.
- **Search Functionality**: Quickly find books by title, author, or genre.
- **Loan Tracking**: Keep track of books loaned to friends and their return dates.

### Tech Stack
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express
- **Database**: MongoDB

### Installation
To get a copy of the project up and running on your local machine for development and testing purposes, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/vyasvihaanv-cyber/Selfbrary.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Selfbrary
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the application:
   ```bash
   npm start
   ```
